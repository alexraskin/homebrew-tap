# Homebrew Tap

Custom Homebrew formulae by [@alexraskin](https://github.com/alexraskin).

## Install

```bash
brew tap alexraskin/tap
```

## Formulae

| Name | Description |
|------|-------------|
| [ata](https://github.com/alexraskin/ata) | Chat with Claude from your terminal |
| [todo](https://github.com/alexraskin/todo-cli) | Simple CLI to format todos |

```bash
brew install alexraskin/tap/ata
brew install alexraskin/tap/todo
```
